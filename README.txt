Front-end & Backend Description:

- Flask has been used in order to interact with Database and Frontend via use of Python.
- Html5 and CSS has been used to design the minimalistic webpage
- SQL has been used to create the tables, MySQLdb is used to insert values in the database
- The same can be done using the STANDALONE Flask app which is in 'backend.py' but 
	an alternate terminal line version has been used to show a different implementation method.
